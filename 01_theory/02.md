# 02. 디자인의 조건

## 디자인의 4대 조건
-합목적성 : 모두가 실용적으로 사용할 수 있는 것, 사물이 일정한 목적에 적합한 방식으로 존재하는 성질, 목적에 맞는 수단을 사용하였는지
-경제성 : 최소의 재료에 의해 최대의 효과를 얻었는지, V(디자인 가치) = P(성능) / C(비용)
-심미성 : 아름다움을 느끼는 미적 의식, 주관적
-독창성 : 창의적인 디자인 조건, 개성적인 미의식 창출, 창조는 기존 디자인을 바탕으로 보다 나은 디자인을 만드는 것

## 굿디자인(Good Design)
-합목적성, 경제성, 심미성, 독창성 + 질서성(4대조건을 하나롭게 조화롭게 설계)

## 리디자인(Re Design)
-다시 디자인, 기존상품의 개선안을 찾아서 새로운 디자인을 개발
